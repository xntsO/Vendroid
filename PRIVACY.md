# Privacy

The current Vendroid FOSS build does not include telemetry or an account system and does not intentionally transmit usage data to the project maintainers.

Vendroid uses Android's USB host APIs to access a USB drive selected by the user and the Storage Access Framework to access files or folders the user explicitly chooses. Installation and image-writing operations process that data locally on the device.

Android, the device vendor, downloaded applications, and linked external websites may have separate privacy practices. Any future build that introduces network telemetry must update this notice and disclose the behavior before release.
